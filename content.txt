似乎有许多新组织出现，在大型行业参与者的支持下，联合构建元宇宙。然后是 M3，一个由艺术家和开源黑客组成的草根社区，他们也想要参与构建开放元宇宙，并且在同一个章程的指导下已经运营了将近 4 年。

M3代表 Metaverse Makers。我们无法决定第三个 M 是什么，所以发挥你的想象力吧。它是 JanusXR、AnarchyArcade、Webaverse 和 Hyperfy 等创新项目背后思想的家园。自 2019 年以来，我们一直致力于构建一个可互操作和去中心化的元宇宙的共同目标，我们的 GitHub 证明了这一点：github.com/m3-org

“策划小组是一种点对点的指导小组，用于帮助成员解决问题、做出决策以及实现个人和职业目标。该小组通常由少数人组成，他们定期会面以相互支持和挑战。成员分享他们的经验，提供建议，并让彼此为进步负责。策划小组的目标是帮助成员利用团队的集体智慧和资源，比他们自己能够更快、更有效地实现他们的目标”——ChatGPT

我们是谁?我们是M3，我们是一群致力于具体化实用“Metaverse”的创客。我们通过开发产品、内容制作、布道和筹款来做到这一点。

元宇宙是一个存在于计算机中的三维世界，由许多有着许多甚至是相互冲突目标的人创建，尽管如此，他们对可互操作的不断发展的标准达成了松散的共识。

所以元宇宙是Matrix吗？也许“黑客帝国”是一种灵感，但我们并不是要重现的现实。

所以元宇宙是互联网吗？网络有很多好的属性，但它不是一个“三维世界”。WebXR 等技术可能会有所帮助。我们接受任何可以提供帮助的技术，只要它基于可互操作的不断发展的标准。

所以元宇宙是VR？AR？2D吗？可以通过许多镜头来观察三维世界。它可以在平面屏幕上，附加到真实位置，或者完全虚拟。互操作性意味着这些都应该可以相互沟通。

所以元宇宙是 Facebook Horizon/Steam/Oculus/Magic Leap？这些项目由设定目标和规定标准的单一企业运营。我们不能在真空中工作，所以我们试图使我们的想法与他们的想法相适应。

所以是 VRChat/Minecraft/Decentraland/Cryptovoxels？这些项目很棒，但互操作性不够，而且通常由一个组织领导。

什么是“创客”？（maker）创客是为实现目标做出贡献的人。创造有多种形式，例如构思、组织、数据研究、设计、执行、资金和其他推进我们使命的努力。创客加入小组不是为了娱乐、欣赏或保持‍“跟上潮流”。如果你不是创客，你仍然可以享受我们开发的产品，并且你仍然可以通过其他方式支持我们的使命。

什么是“实用”？实用意味着我们是有远见的实用主义者。我们的工作不可能存在于真空中。

以下是一些M3不相信的事情：我们是对的，其他人都是错的、一切都应该“开源”、如果我们做出伟大的事情，人们会来帮忙、结果有机地出现，通过我们工作以外的东西、我们感谢免费工作的志愿者、人和组织都是仁慈的、如果我们等待，它就会发生

以下是M3确实相信的一些事情：偏向于行动和暂时的答案，而不是永久的问题和讨论、思路清晰很重要、我们的想法会与持有“异见”巨头想竞争、我们需要钱才能生存、每个人都想要不同的东西，但我们可以一起聚焦、推广我们的想法是一项重要的工作、时间至关重要，因为其他人不会等待

M3的工作：定期开会讨论和提出我们的想法，并对这些想法的进展负责。研究活动既可以学习又可以推广我们的想法。以视频、博客文章、社交媒体活动等形式清晰地传达我们的想法。筹款工作，包括 GitHub Sponsors/Patreon/Venture Capital，其中可用资金有助于实现团队目标。与不一定同意我们观点的其他领导人组织和协调演讲。管理事务，例如 GitHub 组织、Discord 频道、视频转录等。
